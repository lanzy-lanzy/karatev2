# Quick Start - Trainee Profile Management

## New Features
Trainees can now:
- View their complete profile
- Update profile information (name, email, phone, address, etc.)
- Upload and change profile pictures
- See profile pictures in match displays

## Accessing Profile Features

### From Dashboard
1. Click **View Profile** button on dashboard
   - Displays complete profile information
2. Click **Edit Profile** button on dashboard
   - Allows editing profile and uploading picture

### From Sidebar Menu
1. Click **My Profile** in the left sidebar
   - Shows your full profile with all information
2. Click **Edit Profile** button within profile page

## Updating Your Profile

### Step 1: Navigate to Edit Profile
- Dashboard → "Edit Profile" button, OR
- Sidebar → "My Profile" → "Edit Profile" button

### Step 2: Update Information
The edit form has three sections:

**Profile Picture**
- Click "Choose Picture" to upload an image
- Supports JPG, PNG, GIF formats
- Recommended: Clear, professional photo
- Current picture shown as preview

**Personal Information**
- First Name
- Last Name
- Email Address
- Phone Number
- Date of Birth
- Address (multiple lines)

**Training Information**
- Weight (in kg) - auto-calculates weight class
- Emergency Contact Name
- Emergency Contact Phone

### Step 3: Save Changes
- Click "Save Changes" button to update
- Success message appears when saved
- Use "Cancel" to discard changes

## Viewing Your Profile

### Profile Page Shows:
- Your profile picture
- Personal details (name, email, phone, address)
- Contact information
- Training information (weight, weight class, belt rank)
- Emergency contact details
- Performance statistics (points, wins, losses, events)

## Profile Pictures in Matches

### Where Profile Pictures Appear:
1. **Upcoming Matches**
   - Shows both competitors with their profile pictures
   - Larger pictures (more visible than before)

2. **Past Match Results**
   - Winner shown with green border
   - Loser shown with gray border
   - Clear visual indication of match outcome

### Fallback Avatar
- If no profile picture uploaded
- Shows initials of first name
- Color-coded by competition (blue/red for upcoming, green/gray for past)

## Tips

1. **Best Profile Picture**
   - Headshot or chest-up photo
   - Clear face visible
   - Good lighting
   - Professional appearance

2. **Information Requirements**
   - Name and email: Used for communications
   - Phone: Emergency contact and event registration
   - Weight: Determines weight class for fair competition
   - Date of birth: Used for age categories (if applicable)
   - Emergency contact: Safety requirement

3. **Privacy**
   - Email and phone fields optional
   - Profile visible to you and administrators
   - Profile picture visible to all match participants

## Troubleshooting

### Picture Not Uploading
- Check file format (JPG, PNG, GIF)
- File size under 5MB
- Try a different browser
- Clear browser cache

### Changes Not Saving
- Check for error messages (shown in red)
- Fill all required fields (marked as required)
- Ensure phone field is valid format
- Try again or refresh page

### Can't See Picture in Matches
- Upload picture first (profile edit page)
- Clear browser cache (Ctrl+Shift+Del)
- Refresh match page (F5)
- Try different browser

## Next Steps

After updating your profile:
1. View it in **My Profile** page
2. See it displayed in **Upcoming Matches**
3. See it in **Past Results** when matches complete
4. Update again anytime from **Edit Profile**
